# Profile
> Clone the `profile_init.sh` shell script directly to get started

```
curl -O https://raw.githubusercontent.com/peledies/profile/master/profile_init.sh | bash
```



## Environment Install

> you can clone the homebrew shell script directly if thats all you need

```
curl -O https://raw.githubusercontent.com/peledies/profile/master/homebrew.sh && chmod +x homebrew.sh
```
